# Virtualizor 魔方财务/魔方V10 插件
# 目前插件还在开发中，只支持最基本的开通，终止，管理功能。请不要在生产环境中使用。

## 安装
1. 下载插件
2. 解压插件到module目录中
3. 本插件文件夹改名成mfvirtualizor

## 已完成功能
* createAccount （开通）
* TerminateAccount （删除）
* Suspend （暂停）
* Unsuspend （解除暂停）
* virtualizor enduser panel SSO登录跳转
* 开，关，重启 虚拟机

## 未完成功能
* 开通时自定义选项
* 升降配
* Log记录
* 重置密码
* 重装系统
* VNC
* Domain Forwarding
* 手动绑定现存的虚拟机到用户账户
